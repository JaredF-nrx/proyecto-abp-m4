# Proyecto JavaScript - Separado por Operaciones

## Estructura
- **1-1.operaciones_matematicas** → Operaciones matemáticas básicas (+, -, ×, ÷)
- **2-2.condicionales** → if/else y switch
- **3-arrays_ciclos** → Arrays, for, while, forEach
- **4-funciones** → Funciones reutilizables y modularización
- **5-object** → Objetos, métodos, arreglos de objetos

## Lo que hace cada archivo
Cada archivo:
- Se ejecuta automáticamente
- Muestra sus resultados en consola
- Funciona independientemente
- Está comentado para entender qué hace
